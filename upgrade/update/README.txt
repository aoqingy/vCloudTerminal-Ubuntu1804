以下是目录说明：

config    :用于配置定制系统的/root/.config
deplist   :用于在定制系统中安装非官方的软件包
firefox   :用于配置定制系统的/root/.mozilla
keybinding:用于实现通过键盘调出终端窗口
logo      :用于定制安装过程中和安装后的LOGO
OEM       :用于某些用户的特殊需求
systemd   :用于实现定制系统的开机运行服务
tools     :
usbsrvd   :用于实现定制系统的USB共享功能
vfd       :用于实现定制系统的终端管理功能
vmware    :用于在定制系统中安装VMware客户端
